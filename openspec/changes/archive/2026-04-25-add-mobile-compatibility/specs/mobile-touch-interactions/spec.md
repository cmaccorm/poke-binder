## ADDED Requirements

### Requirement: Tap to assign card
The system SHALL allow single-tap to assign a searched card to a slot.

#### Scenario: Tap empty slot opens search
- **WHEN** user taps empty slot
- **THEN** card search modal opens for that slot

#### Scenario: Tap filled slot shows details
- **WHEN** user taps filled slot with card
- **THEN** card detail modal opens showing card information
